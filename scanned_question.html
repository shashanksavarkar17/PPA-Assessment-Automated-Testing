Breakup Story Difficulty: HARD | Max Score: 75 Partition always feels like curse. Countries, houses and sometimes people too get separated and that feeling is heavy for the heart. But still, they move on and get to their daily life routine. Alice is going through the breakup phase and wants to solve more and more partitioning problems. Can you please help to solve the problem below: Given an array A of N positive integers, you must partition the array into exactly K contiguous, non-empty subarrays. The cost of a single subarray spanning from index L to R is defined as the square of the sum of its elements: COST(L,R)= (AL+AL+1+.....+AR)2 The total cost of the partition is the sum of the costs of the K subarrays. Find the minimum possible total cost. Input Format: The first line contains two space-separated integers, N (the number of elements in the array) and K (the number of contiguous subarrays to form). The second line contains N space-separated integers, representing the elements of the array A. Output Format: Print a single integer representing the minimum possible total cost to partition the array into exactly K contiguous subarrays. Example: Input: 4 2 1 2 3 4 Output: 52 Explanation: We need to split the array [1, 2, 3, 4] into exactly 2 segments. The optimal partition is [1, 2, 3] and [4]. Constraints: 1 <= N <= 105 1 <= K <= 50 1 <= Ai <= 100

Language: C++
